  THE INCANTATION:

    npm install -g @anthropic-ai/claude-code

  Let's break this down — every piece means something:

    npm                         — the program you're running.
                                  Remember npm? It came bundled
                                  with Node.js, which you installed
                                  earlier. Think of it as an
                                  APP STORE for developer tools.

    install                     — what you want npm to do.
                                  (Download and set up a tool.)

    -g                          — a FLAG that means "global."
                                  Install it so it's available
                                  everywhere on your Mac, not
                                  just in this folder.

    @anthropic-ai/claude-code   — the PACKAGE NAME. The specific
                                  tool you're installing, made by
                                  a company called Anthropic.

  Run the command now. You'll see text scrolling by — that's npm
  downloading the tool and all its pieces. This is normal!

  TROUBLESHOOTING

  If you see an error like:

    dyld: symbol not found ... expected in /usr/lib/libc++.1.dylib

  This means your version of Node.js doesn't match your macOS.
  The fix: reinstall Node with a version compatible with your Mac.

  Run this to check if Node works:

    node -e "console.log('works')"

  If it prints "works" — you're good. If it throws an error,
  you'll need to reinstall Node before continuing.
