  Run this command:

    xcode-select --install

  A window will pop up on your screen — click "Install."

  What does this do?

  This is a SYSTEM COMMAND — it talks directly to your Mac's
  operating system and asks it to install developer tools.

  One of those tools is called 'git'. You don't need it right
  now, but later it will let you save snapshots of your work —
  like save points in a game.

  The download will run in the background while you keep
  exploring. No need to wait — move on to door-2!
