  CLAIM YOUR REWARD!

  One more concept. Every terminal command works like this:

    command   argument
    ───────   ────────
    cat       open-me-first.txt    (read THIS file)
    cd        door-1               (go to THIS folder)
    afplay    .treasure.wav        (play THIS sound)

  The COMMAND is what to do.
  The ARGUMENT is what to do it to.

  'afplay' is a program already on your Mac — it plays audio.
  '.treasure.wav' is the hidden sound file in this room.

  Type:  afplay .treasure.wav
